
# Acesso Tech

Projeto educacional e de acessibilidade criado para ajudar pessoas com deficiência auditiva e estudantes iniciantes a aprender tecnologia de forma mais visual, simples e inclusiva.

## Objetivo

Criar materiais gratuitos e, no futuro, ferramentas acessíveis para apoiar o aprendizado de tecnologia e programação.

## Público-alvo

Pessoas com deficiência auditiva, estudantes iniciantes em tecnologia e jovens que enfrentam dificuldades com conteúdos baseados principalmente em áudio.

## Tecnologias previstas

* HTML
* CSS
* JavaScript
